# Password Strength Checker

## Description
This is a simple Python script that checks whether a password is strong or weak based on common password security rules.

## Features
The program checks if the password contains:

- Minimum 8 characters
- At least one uppercase letter
- At least one lowercase letter
- At least one digit
- At least one special character

## Technologies Used
- Python
- String methods
- Conditional statements
- Loops

## How to Run

1. Open terminal in the project folder
2. Run the script in python environment
3. Enter the desired Password to check its strength
4. The script will classify the password as Strong or Weak.
