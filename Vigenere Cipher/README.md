# Vigenere encoder/decoder

</div>

### Description
This is a simple python script to encode/decode vigenere ciphers

### How To Use
0. ensure that you have Python installed
0. clone or download the script file to your computer
0. open a terminal or command prompt
0. navigate to the directory where the script is located
0. run the script by executing the following command:
```
python vigenere.py
```
you should get a prompt asking whether you want to use the full ascii character set (' ' to '~') or the standard 26 letter alphabet:
```
ascii (default)/standard (26 letter) > ascii
```

you should then get a prompt asking whether you want to encode a string or decode a string:
```
encode (e)/decode (d) > e
```
then enter your string:
```
string to encode > Hello, World!
```
then you will be asked to enter a key:
```
enter key > vigenere
```
then you will get the appropriate output:
```
output: ?OTR^eJUiVLf
```
